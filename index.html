Welcome to SoundCloudDB

SoundCloudDB is a database with popular Artists, Albums, and Songs. Click Any of the buttons located above to access the database catalogs. Each Catalog has a search function that allows you to search by Genre, Artist, Song or Album name.

Have songs you want to share?

The links provided below on the following table will take you to pages that will allow you to enter Artists, Albums, and Songs into the database. In order to properly add songs to the database you must first enter Artist, then Album, and song lastly. Each Artist and Album entry generates a unique ID you must use for album and song entries. These are displayed in the Artist and Album catalogs.


Steps Below:

  1. Insert an Artist
  2. Insert an Album
  3. Insert a Song
